How to use it (WIP)

For now, you can just clone this repo and...

  1. Npm or Yarn install its deps: npm install
  2. Build your own application you wanna test and serve it on a local port (i.e. 9999)
  3. Use angular.spec.js as an example
  4. Run npm test. This will run the perf compain tests
  5. All reports will be generated in the public folder
  6. Run a local server to serve the public folder
  7. Enjoy üéâ

Perfy will then merge all those files into a single file named paint-with-web-component.json. This file will eventually contain all samples from each timestamped files. It will also update the index.html to load that new file paint-with-web-component.json (or any other merged file, of course).

So, you just need to serve the public folder, run your perf compaigns and after each compaign you can refresh the perfy UI to view the updated generated data from Benchpress.
